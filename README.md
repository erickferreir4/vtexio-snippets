# VTEX IO Snippets

A collection of vtex io snippets for faster development.

![HTML gif](https://user-images.githubusercontent.com/67771985/131255337-d53f3408-b60d-44a2-93ba-9a3240a7436e.gif)

### Install

Use your plugin manager of choice, e.g.

```lua
-- Packer
use "erickferreir4/vtexio-snippets"

-- Plug
Plug 'erickferreir4/vtexio-snippets'

-- If you're using coc.nvim, you can use:
CocInstall https://github.com/erickferreir4/vtexio-snippets@master
```
